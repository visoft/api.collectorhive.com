# CollectorHive API

## Introduction

CollectorHive is a RESTful API for managing CollectorHive data.

## Development

1. Copy `env.template` to `.env`

```
cp env.template .env
```

2. Customize data in `.env`
3. Run `npm install`
4. Run `npm run dev`

## Usage

1. Visit `http://localhost:8000` (or whatever `CH_PORT` you specified in `.env`)
